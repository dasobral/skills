Why reorganize

Three equal export paths

Cursor, Claude, and Codex each own scaffolding under adapters/ and generate committed plugins under plugins/. Core never holds platform scaffolding.

Sync all: ./bin/skills-export sync all

./bin/skills-maintain
Periodic maintain

Cron is for landing drops

Not for installing into ~/.claude or Cursor — that is ./bin/skills-install.

  1. 1
    Configure basics Copy config.env.example or pass flags: repo path, schedule, python, log.
  2. 2
    Install crontab ./scripts/cron/install.sh writes a marked block with SHELL + PATH.
  3. 3
    Wrapper runs maintain skills-maintain.sh cds to the repo, checks PyYAML, appends landing/maintain.log.
  4. 4
    You still commit Cron updates files on disk. Review git status, then commit — or use CI.
# preview then install
./scripts/cron/install.sh --dry-run
./scripts/cron/install.sh --schedule '0 */6 * * *'
./scripts/cron/uninstall.sh
