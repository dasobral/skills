## Register Quick-Check

Before delivering, confirm:

- [ ] Opening sentence is a concrete hook (not "I am writing to apply for…")

- [ ] No QKD framing errors ("quantum software", "quantum hardware")
- [ ] No forbidden phrases ("passionate", "leverage", "synergy", "cutting-edge")
- [ ] Audience register correct (academic / industry / hybrid)
- [ ] Word/character count reported if a limit applies
- [ ] No invented publications, roles, or credentials
