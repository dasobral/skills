# Crypto Change Radar

Claude Code plugin generated from portable core.

## Skills

- `build-crypto-inventory`
- `review-crypto-delta`
- `plan-pqc-migration`
- `test-crypto-interoperability`

## Install

From the skills repo marketplace:

```text
/plugin marketplace add .
/plugin install crypto-change-radar@dasobral-skills
```

Or copy this directory into a Claude Code plugins path.
