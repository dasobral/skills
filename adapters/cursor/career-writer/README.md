# Career Writer

Professional career documents with audience-aware framing.

## Skills

| Skill | Purpose |
|-------|---------|
| `career-documents` | CVs, cover letters, statements, proposals |

## Setup

Copy `skills/career-documents/references/profile-template.md` to `references/my-profile.md` and fill in your details.

## Agents

- `document-editor` — edits documents against your profile

## Rules

- Never invent credentials
- Track word/character limits
- Frame content per audience (academic vs industry vs government)
